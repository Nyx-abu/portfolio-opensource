# Portfolio Brief — Master Reference

## Identity
Personal portfolio. Target audience: recruiters, founders, designers.
Goal: communicate operating at a senior/exceptional level.

## Design Language
Cinematic · editorial · typography-first · minimal · emotionally designed
References: Apple, Arc Browser, Linear, Nothing, Vercel, Framer, Stripe Press

## Palette
- Primary: black / off-white / grayscale
- Accent: muted cobalt (#3B5BDB range — one accent, used sparingly)
- NO: neon, gradients, glassmorphism, floating blobs, cyberpunk

## Typography Rules
- Local display fonts (DxPlayhigh, Kiya Handwrite) from /fonts; pair with a system sans for body
- Oversized type moments allowed
- Whitespace is luxurious and intentional
- Typography carries personality

## Motion Philosophy
- Physical, intentional, layered, cinematic
- NO: random floats, parallax spam, annoying entrances
- Framer Motion primary, GSAP only if absolutely required

## Banned Patterns
- Generic hero "Hi I'm X" centered layout
- Emoji anywhere
- Progress bar skills
- Bento cards without intention
- Template sections
- Glassmorphism
- Gradient spam

## Tech Stack
Next.js 15 (App Router) · TypeScript · TailwindCSS · Framer Motion
Lenis (smooth scroll) · PostgreSQL · Prisma · NextAuth v5
Server Actions · Edge-optimized · shadcn-style primitives where useful

## Performance Targets
Near-perfect Lighthouse · instant feel · optimized fonts
Minimal JS hydration · lazy load · route-based code splitting · zero jank
